Anyways, Now I have to make the structure of the language. The subject goes first, then the verb, then the object. That way, the subject or object can be left out, and you can still find what the other is. It's also a common sentence order.

This is the first sentence I made! There are some things to take from this:
  1. A "t" at the end makes it a noun.
  2. Similarly, An "y" makes it a pronoun. The reason for this might be explicated?
  3. The copula is extremely simple: "s". No vowels!
  4. The "`" makes a noun a proper noun. This can be ommited if unambiguous. But if it isn't, absense makes it improper. Maybe this can apply to other things?
  5. A ";" ends the sentence. This can really be any other one if in written form, since there w,ould, the.n, be .n,o pesky peri.ods.

Since that was the first sentence, there were obviously a lot of things to take from it. The effect of finishing? this conlang is that there will be a loss of things to talk about later.

Still some to cover.
  1. An "s" at the end makes it a verb.
  2. The vocabulary is based off of Indo-European.
  3. The structure for verbs is usually "cvcvcvs". The last two vowels have definition for not being there, but the rest is mandatory.
  4. The first vowel is about the tense of the verb. Currently, we know "i" is simple past tense. "_" is simple present tense.
  5. The second vowel is about the aspect of the verb. Currently, we know "a" is perfective tense. No vowel means that the aspect is not given.
  6. The third vowel is about the mood/modality of the verb. No vowel means that the mood is not given.
